# flask-setup-tutorial-Miniconda-
Building a minimal Flask API to understand the environment setup.

Goal:
1. Create a clean Flask environment  
- Activate  
- Install packages (Flask)  
- Folder:  
flask-setup-practice  
│  
├── app.py  
└── requirements.txt  

2. Run a minimal Flask server
 - Write /health  
 - Run server  

3. Freeze dependencies
 - pip freeze > requirements.txt

4. Verify the setup works
- deactivate and reactivate -> run app.py
